Política de Privacidade

Esta é a política de privacidade para a API de Processamento de Excel.

Coleta e Uso de Dados

Nossa API recebe ficheiros Excel enviados pelo utilizador através de uma Ação do GPT Builder com o único propósito de processá-los e retornar os dados em formato JSON.

Os ficheiros são processados em memória e guardados temporariamente no servidor apenas durante o tempo necessário para o processamento. Após a conclusão do pedido, os ficheiros são permanentemente eliminados.

Armazenamento e Segurança

Não armazenamos permanentemente nenhum ficheiro ou dado extraído dos ficheiros enviados. A segurança dos dados durante a transmissão é garantida pelos protocolos HTTPS da Vercel e da OpenAI.

Partilha de Dados

Não partilhamos os dados dos seus ficheiros com terceiros.

Contacto

Se tiver alguma dúvida sobre esta política de privacidade, pode entrar em contacto.
